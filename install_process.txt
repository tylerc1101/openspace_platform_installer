Step 1: Configure IDRAC of servers for remote access
Step 2: Configure BIOS of each servers
Step 3: Install latest RHEL on server through CD
Step 4: Once OS is installed, Add simple IP to servers so that they can be reached
Step 5: sudo su         # On management server
Step 6: mkdir /opt/kratos; chown kratos:kratos /opt/kratos
Step 7: scp OpenSpace Installer Platform in /opt/kratos
Step 8: scp basekit-1.0.1.tar in /opt/kratos
Step 9: scp latest onboarder in /opt/kratos/openspace_platform_installer/images
Step 10: tar -xvf /opt/kratos/basekit-1.0.1.tar -C /
Step 11: cd /opt/kratos/openspace_platform_installer
Step 12: Copy sample env to a new uniquely named env dir
Step 13: Configure environment/<env>/config.yml
Step 14: export TMPDIR=/var
Step 14: python onboarder-run.py (probably should run from the Idrac session at least at first because of the drop connection)
